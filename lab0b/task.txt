Напишите программу, которая будет принимать в качестве аргумента имя текстового файла, 
и выводить CSV файл (http://ru.wikipedia.org/wiki/CSV) с колонками:

1. Слово.
2. Частота.
3. Частота (в %).
CSV файл должен быть упорядочен по убыванию частоты, то есть самые частые слова
должны идти в начале. Разделителями считать все символы кроме букв и цифр.


Аргументы командной строки программы:
> word_count.exe input.txt output.csv


------------- Решение
1. Считать слова из файла в буфер.
2. Пока буфер не пуст.
    2.1 Увеличиваем счётчик в map для этих слов.
3. Когда буфер кончился, то закрываем файл.
4. Создаём CSV файл и в него записываем мапу.

Классы:
1. Буфер слов. Открывает файл и возвращает буфер слов 
    Методы: isEmpty(), update, readFile_(), 

2. Счётчик слов. Принимает массив слов, подсчитывает слова. Сохраняет в  мапу
    Методы: countWords(string[] words, map wordsCount),

3. Сохранятель в CSV
    Методы: save, saveToCSV,
